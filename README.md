## 🌈 코로나 런 🕰
#### 코로나 19 감염 및 백신 접종 현황 확인 앱 📟
> 4학년 1학기 협업하는 프론티어 결과물 📔

<br>

#### 📚 개요
> 코로나19 백신에 대한 정보와 이를 조회 및 예약 하기 위한 수단을 제공하고, 지역별 접종률, 감염률을 정보를 표시해줍니다.

``` 
+ 예방 접종 센터 확인, 백신 현황 조회 및 예약
+ 확진자(감염률)+ 백신 접종률 확인 -> 지역 간 비교
+ 코로나 19 예방수칙 제공  
+ 백신 정보 제공
```

</div>

#### :school_satchel: 사용 API
```
* 질병청 코로나19 감염 현황
* 질병청 백신 센터 현황
* 질병청 백신 접종 현황
```
